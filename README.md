## Description

ZRAIL is a responsive, frontend-only railway ticket booking website that simulates core railway reservation functionalities. The project demonstrates user authentication, train management, search, and booking workflows using a clean and intuitive interface.

## Project Overview

This application represents a simplified railway booking system where users can search and book train tickets, while administrators can manage train details. The project focuses on frontend logic, UI/UX design, and role-based access.

## Features

### User Module

* Login authentication
* Search trains by train number, train name, source station, and destination station
* View train schedules and availability
* Book railway tickets

### Admin Module

* Secure admin login
* Add new trains
* Edit existing train details
* Delete trains from the system
